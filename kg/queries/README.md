# CLI-запросы к графу знаний Dota 2

В директории `kg/queries` находятся два CLI-скрипта для запросов к RDF-графу знаний предметов Dota 2.

---

## `full_recipe.py`

Показывает **полный рецепт предмета**, разложенный до базовых компонентов
(учитывает вложенные рецепты и количества).

**Пример:**

```bash
poetry run python -m kg.queries.full_recipe "moon shard"
```

**Результат:** список базовых предметов и их количество.

---

## `crafts_into.py`

Показывает, **в какие предметы может входить указанный предмет**
(прямо или транзитивно, как часть рецепта).

**Пример:**

```bash
poetry run python -m kg.queries.crafts_into "staff of wizardry"
```

**Результат:** список всех предметов, которые можно скрафтить с его использованием.

---

## Общее

* Источник данных: RDF-онтология `kg-dota.rdf`
* Поиск предметов нечувствителен к регистру и поддерживает опечатки
* Используется `DotaKgBuilder` для загрузки и обхода графа
